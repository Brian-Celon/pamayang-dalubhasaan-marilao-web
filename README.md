# Pambayang Dalubhasaan Marilao - School Website

<h2>Introduction</h2>
Pambayang Dalubhasaan ng Marilao currently does not have an official website. This project aims to develop a modern, responsive, and user-friendly website that serves as the school's official online presence. The website will provide prospective students, current students, alumni, parents, faculty, and the community with access to essential information, including the school's history, academic programs, facilities, news, events, and contact details. It is also designed to support future features such as a student portal, online inquiries, and administrative content management. This project also serves as a practical application of modern web development practices, with a focus on responsiveness, accessibility, security, maintainability, and scalability.

<br>Note: This project is for practice purposes only

<h3>Tech Stack</h3>
- <b>Frontend:</b> HTML5, CSS (Tailwind CSS), JavaScript <br>
- <b>Backend:</b> Node.js, Express.js <br>
- <b>Tooling:</b> Git, GitHub
